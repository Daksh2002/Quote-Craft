## Setup

1. Clone this repo:
   ```bash
   git clone <repo-url>
   cd QuoteCraft
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Launch the server:
   ```bash
   python app.py
   ```

4. In your browser, go to:
   ```
   http://127.0.0.1:5000
   ```
